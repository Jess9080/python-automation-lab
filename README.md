# Python Automation Lab

Repositorio con scripts en Python orientados a automatización de tareas y gestión del sistema.

## Scripts incluidos

### 📂 file_organizer.py
Organiza archivos automáticamente en carpetas según su extensión.

### 🖥️ system_report.py
Muestra información básica del sistema:
- Sistema operativo
- Versión
- Arquitectura
- Usuario
- Nombre del equipo

## Tecnologías
- Python

## Objetivo
Aplicar automatización en tareas reales de gestión de archivos y análisis del sistema.

## Ejecución
```bash
python file_organizer.py
python system_report.py
